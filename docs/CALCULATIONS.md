# FleetFlow Calculations

- **Fuel Efficiency**: `(odometerEnd - odometerStart) / liters`
- **Utilization Rate**: `(Active Vehicles + Maintenance Vehicles) / Total Non-Retired × 100`
- **Cost per KM**: `(Total Fuel + Total Maintenance) / Total Distance Driven`
- **Driver Completion Rate**: `(Completed Trips / Total Assigned Trips) × 100`
